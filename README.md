## Menu Lateral (Sidebar) — HTML, CSS e JavaScript

Projeto simples de **menu lateral responsivo** com estado **expandido/recolhido**, feito com **HTML + CSS + JavaScript**.

### Funcionalidades

- **Sidebar recolhível**: botão alterna a classe `open-sidebar`
- **Estados visuais**: item ativo e hover
- **Layout**: navegação fixa à esquerda e conteúdo no `main`

### Estrutura do projeto

- **`index.html`**: marcação da sidebar e do conteúdo
- **`style.css`**: estilos do menu (animações/expansão)
- **`script.js`**: toggle da classe `open-sidebar`
